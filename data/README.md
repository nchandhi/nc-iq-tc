# Sample Documents

This folder contains sample PDF or text documents that will be uploaded to Azure AI Search.

## Adding Your Own Documents

1. Place PDF files (`.pdf`) or text files (`.txt`) in this folder
2. Run `python scripts/01_upload_data.py` to process and upload them

## Sample Content

For the lab, you can use any documents. Example topics:
- Product catalogs
- Policy documents  
- FAQ sheets
- Knowledge base articles
