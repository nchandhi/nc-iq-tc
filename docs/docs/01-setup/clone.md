# 1.1 Clone Repository

Open a terminal and clone the lab repository:

```bash
git clone https://github.com/nchandhi/nc-iq-tc.git
cd nc-iq-tc
```

## Repository Structure

```
nc-iq-tc/
├── infra/              # Bicep infrastructure files
├── scripts/            # Python lab scripts
├── data/               # Sample documents
├── evals/              # Evaluation data
└── docs/               # This documentation
```

!!! tip "Using Codespaces?"
    You can also open this repo directly in GitHub Codespaces for a pre-configured environment.
